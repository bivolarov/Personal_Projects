From text to labyrinthe¶

Permet d’importer un labyrinthe sous format texte en un labyrinthe de notre classe Maze.

Function Description¶

from_text2laby.from_text2laby(file)[source]¶

return the maze created from the path of the maze described in the file

Returns:a maze
Return type:Maze
UC:The name of the file must exist
Examples: 
>>> from_text2laby("maze_5_5_0.txt")
